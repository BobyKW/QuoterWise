# **App Name**: QuoterWise

## Core Features:

- Secure User Authentication & Profile Management: User registration, login, password recovery via email, and management of a professional company profile (business name, tax ID, address, contact, optional logo, and default terms). Ensures private access to quotes.
- Comprehensive Quote Lifecycle Management: Full CRUD operations for quotes: create, list, view details, edit, duplicate, and delete. Includes automatic, configurable quote numbering, client details, structured sections with detailed line items, and automated calculations for subtotals, discounts, taxes, and final totals.
- Flexible Template & Reusable Block System: Users can create and edit custom quote templates and save reusable 'blocks' of items. Includes pre-loaded industry-specific templates (e.g., Construction, Plumbing, Web Design) to kickstart quote creation.
- AI-Powered Item Description Tool: Utilizes AI to generate clear and concise descriptions for quote items or sections based on concept keywords, industry, and desired detail level, streamlining the quote creation process.
- Quote Status Tracking & Activity Timeline: Manage quotes through defined statuses (Draft, Sent, Accepted, Rejected, Negotiating, Expired) and maintain a simple activity log for key events (creation, edits, status changes). Includes filters and search for efficient quote overview.
- Professional PDF Export & Secure Sharing: Generate branded PDF documents of quotes, including company details, client information, itemized breakdowns, totals, and general terms. Provides an optional secure, read-only shareable link for clients.

## Style Guidelines:

- The app uses a light color scheme designed for 'Calm Professionalism' and 'Organized Clarity'. The primary color, used for interactive elements and highlights, is a soft slate blue (#5C89AE). The background color is an almost imperceptibly light cool gray (#F5F6F8) to ensure high readability and a clean interface. The accent color, a vibrant aqua (#3AD2DD), is used sparingly for call-to-action elements or to draw attention to critical information, providing a modern contrast without overwhelming the professional aesthetic.
- The app uses 'Inter' (sans-serif) for all text elements, including headlines and body content. 'Inter' was chosen for its excellent legibility across various screen sizes, modern, objective appearance, and versatility, aligning perfectly with the professional and efficient nature of the application.
- Minimalist and clean line icons are utilized throughout the application. This approach ensures clarity and complements the professional, uncluttered dashboard interface.
- The interface features a responsive 'dashboard' style layout with intuitive navigation, likely a sidebar or top navigation. Content areas are clearly defined for list views, detailed forms, and individual item displays, optimized for efficient information access and input. Drag-and-drop functionality will be implemented for intuitive reordering of sections and line items.
- Subtle and functional animations are incorporated to enhance the user experience, particularly for state transitions (e.g., status updates, form submissions, or item reordering) and smooth navigation between different views. These animations are designed to be non-intrusive and provide visual feedback.
